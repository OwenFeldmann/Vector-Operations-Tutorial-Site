Point-Point Subtraction

A vector v can be calculated by subtracting a start point P from an end point Q. 

	v = Q-P = [Q1-P1, Q2-P2, ... , Qn-Pn]

The terms in this equation can be worked with to show that a vector can be added to a start point to result in an end point. 

	Q = P + v

Tutorial Use

On the canvas on the left, click a position to define a start point. Then click to define an end point. A vector will be drawn and information on the points and vectors will be displayed. Clicking a third time will reset the tutorial and define the position of a new start point.
